CSS

  • Margin: indicates the space we want around the outside of an element
  • Padding: indicates how much space we want around the content inside an element
  • the dot selector is for a class
  • Classes are best for reusable styling
  • the hash selector is for an ID
  • Use ID selector for unique identification and more specified element selection OR creating anchor links

  • for loops are used to iterate over a collection of data and return whatever you ask for in the index

  • 1. A variable (the first part of your forloop) to keep track of how many times the code executes and what index too start at ex x=0
  • 2. A boolean is used with variablename.length ... As long as the variable is less than the array, it will continue to iterate.
  • 3. Finally, inside curly brackets, define the code you want to repeat //ex: { console.log(shapes)[x]}

  • control flow dictates that JavaScript files are executed from top to bottom, order matters.
